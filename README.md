# djalmaaraujo/homebrew-tap

Homebrew tap for [djalmaaraujo](https://github.com/djalmaaraujo)'s tools.

## Available

### [piper](https://github.com/djalmaaraujo/piper) — stream any command's output live over HTTP

```bash
brew install djalmaaraujo/tap/piper
```

Casks are published automatically by [GoReleaser](https://goreleaser.com) on each release.

### [claude-usage-menubar](https://github.com/djalmaaraujo/claude-usage-menubar) — Claude Code quota, live in the menu bar

```bash
brew install djalmaaraujo/tap/claude-usage-menubar
```

Native macOS menu bar app. Reads usage straight from the `claude` CLI — no browser cookie, no login of its own. Checks for updates on its own and can self-update via `brew upgrade`.
