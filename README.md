# djalmaaraujo/homebrew-tap

Homebrew tap for [djalmaaraujo](https://github.com/djalmaaraujo)'s tools.

## Available

### [piper](https://github.com/djalmaaraujo/piper) — stream any command's output live over HTTP

```bash
brew install djalmaaraujo/tap/piper
```

Casks are published automatically by [GoReleaser](https://goreleaser.com) on each release.
